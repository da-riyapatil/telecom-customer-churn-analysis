# Power BI Service Features

## Overview
After building the dashboard in Power BI Desktop, Power BI Service can be used to publish and share the report online.

For this project, Power BI Service is useful for making the churn dashboard easier to access, review, and present.

## Features Relevant to This Project

### 1. Report Publishing
The Power BI report can be published from Power BI Desktop to Power BI Service.

This allows the dashboard to be viewed online without opening the `.pbix` file every time.

### 2. Dashboard Sharing
Power BI Service makes it easier to share the dashboard with others through a report link or workspace access.

This is helpful when presenting project work to recruiters, mentors, or peers.

### 3. Interactive Filtering
The published report keeps the interactive filtering and slicer experience available in the online version.

Users can explore churn by:
- contract type
- payment method
- internet service
- tenure group
- churn category

### 4. Workspace Organization
Power BI Service helps organize reports and related content inside a workspace.

This makes it easier to manage and access the project dashboard in one place.

### 5. Scheduled Refresh
If the source data is updated, Power BI Service can support scheduled refresh so the dashboard stays updated.

For this project, this feature is useful to understand from a practical point of view, even if the dataset is static in the portfolio version.

### 6. Web Access
The dashboard can be accessed through a browser, which makes it easier to showcase the project without requiring Power BI Desktop on every system.

## Why It Matters
Power BI Service adds practical value to a dashboard project because it supports:
- online access
- report sharing
- interactive exploration
- easier presentation of insights

## Project Note
In this project, the main work was done in **Power BI Desktop** using data preparation, DAX, and dashboard design.

Power BI Service is an additional platform feature that can be used to publish and present the final report more effectively.
